# 9to5 University

Free, anonymous workplace education and career tools.

## Mission
To empower employees with knowledge of their rights and career growth tools
without requiring signups or compromising privacy.

## Monetization
- Advertising
- Affiliate partnerships
- Employer sponsorships
- Grants & NGOs

## Tech
- Next.js 14
- Tailwind CSS
- Deployed on Vercel

## Privacy
No accounts. No stored personal data.
